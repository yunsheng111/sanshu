# Common 通用规划

存放不属于特定工作流的通用规划和分析文档。

## 目录结构

- `wip/` - 临时工作文件（自动清理 > 30 天）
  - `research/` - 分析/检索过程记录
  - `planning/` - 规划过程记录
  - `execution/` - 执行过程记录
- `plans/` - 正式计划文件
- `reviews/` - 审查报告
- `progress/` - 进度追踪
- `archive/` - 已归档文档

## 用途

存放跨工作流的通用文档，如：
- 技术调研报告
- 架构决策记录（ADR）
- 性能优化方案
- 安全审计报告
